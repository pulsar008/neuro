Protection de la vie privée

Informations générales

L'information qui suit donne un aperçu simple de ce qu'il advient de vos renseignements personnels lorsque vous visitez notre site Web. Les données personnelles sont toutes les données avec lesquelles vous pouvez être personnellement identifié. Si ce qui suit se réfère aux "clients", il s'agit de l'exploitant du site Web - dans la mention d'impression également désigné comme "éditeur responsable".


Qui est responsable de la collecte des données sur ce site ?

Le traitement des données sur ce site Web est effectué par l'exploitant du site Web. Vous trouverez ses coordonnées dans les mentions légales de ce site web.


Comment recueillons-nous vos données?

Vos données seront collectées, d'une part, par vous en nous les communiquant. Il peut s'agir, par exemple, de données que vous saisissez dans un formulaire de contact.
D'autres données sont automatiquement collectées par nos systèmes informatiques lorsque vous visitez le site Web. Il s'agit principalement de données techniques (p. ex. navigateur Internet, système d'exploitation ou heure de l'appel de page, la langue). Ces données sont collectées automatiquement dès que vous entrez sur notre site Web.


À quoi servent vos données?

Une partie des données est collectée pour s'assurer que le site Web est exempt d'erreurs. D'autres données peuvent être utilisées pour analyser votre comportement d'utilisateur. Votre adresse IP est stockée par nos serveurs pendant 24 mois. Ceci est fait pour des raisons de sécurité (le logiciel de pare-feu "Fail2Ban" l'utilise pour repousser de manière fiable les attaques sur nos systèmes serveurs). Après un maximum de 24 mois (ou demande écrite correspondante), les adresses IP seront supprimées.


Quels droits avez-vous sur vos données?

Vous avez le droit à tout moment et gratuitement de recevoir des informations sur l'origine, le destinataire et la finalité de vos données personnelles enregistrées. Vous avez également le droit de demander la rectification, le blocage ou la suppression de ces données. Vous pouvez nous contacter à tout moment à l'adresse indiquée dans les mentions légales pour cette question et d'autres questions relatives à la protection des données. En outre, vous avez le droit de faire appel auprès de l'autorité de surveillance compétente.


Outils d'analyse et outils tiers

Lorsque vous visitez notre site Web, votre comportement de navigation peut être évalué statistiquement. Cela se fait principalement à l'aide de cookies et de programmes dits d'analyse. L'analyse de votre comportement de surf est généralement anonyme ; le comportement de surf ne peut pas être retracé jusqu'à vous. Vous pouvez vous opposer à cette analyse ou l'empêcher en n'utilisant pas certains outils. Vous trouverez plus de détails dans notre politique de confidentialité sous la rubrique "Modules tiers et outils d'analyse". Vous pouvez contredire cette analyse. Nous vous informerons des possibilités d'opposition dans cette déclaration de protection des données.

Nous traitons vos données personnelles de manière confidentielle et conformément aux dispositions légales en matière de protection des données ainsi qu'à la présente déclaration de protection des données.

Lorsque vous utilisez ce site Web, diverses données personnelles sont collectées. Les données personnelles sont des données permettant de vous identifier personnellement. Fichiers journaux du serveur

Le fournisseur des pages recueille et stocke automatiquement des informations dans des fichiers journaux du serveur, que votre navigateur nous transmet automatiquement. La base du traitement des données est l'art. 6 al. 1 let. b ODSGVO, qui permet le traitement des données pour l'exécution d'un contrat ou de mesures précontractuelles.
